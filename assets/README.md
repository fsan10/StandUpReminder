# 图标文件占位符

此文件夹用于存放应用图标文件。

## 需要的图标文件

- `icon.png` - 托盘图标 (建议 16x16 或 32x32 像素)

## 图标来源建议

1. 从 [Flaticon](https://www.flaticon.com/) 下载免费图标
2. 使用 [Iconfont](https://www.iconfont.cn/) 阿里巴巴矢量图标库
3. 自己设计或使用 AI 生成

## 推荐图标关键词

- hourglass, timer, clock, health, reminder, stand up
- 沙漏, 计时器, 时钟, 健康, 提醒, 站立

## 格式要求

- 格式: PNG (支持透明背景)
- 尺寸: 16x16, 32x32, 64x64
- 颜色: 建议使用白色或浅色图标，适配深色任务栏

---

**注意**: 如果没有提供图标文件，应用仍能正常运行，但系统托盘功能将被禁用。
