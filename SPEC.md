# SPEC.md - StandUpReminder 设计文档

## 项目概述

**名称**: StandUpReminder  
**目标**: 帮助久坐办公的用户定时休息，预防腰椎疾病  
**平台**: Windows PC  
**技术栈**: Electron + Node.js

---

## 核心功能

### 1. 久坐监控
- 使用 Windows API `GetLastInputInfo` 检测系统空闲时间
- 空闲时间 = 距离上次鼠标/键盘/任何输入的时间
- 监控周期可配置（默认：45分钟）

### 2. 强制休息锁屏
- 工作时长到达后，屏幕逐渐变灰（渐变动画，2秒）
- 弹出全屏遮罩层，显示：
  - 倒计时（分钟:秒）
  - 沙漏动画
  - 健康科普提示（随机轮换）
- 完全屏蔽鼠标和键盘事件
- 休息时长可配置（默认：5分钟）

### 3. 休息结束
- 倒计时归零后，自动关闭锁屏
- 恢复鼠标键盘操作
- 重新开始新一轮监控

### 4. 系统托盘
- 最小化到托盘运行
- 右键菜单：设置、暂停、退出
- 左键点击显示主窗口

### 关键API

**检测空闲时间（Node.js addon）：**
```javascript
// 使用 node-ffi-napi 调用 Windows API
const ffi = require('ffi-napi');
const user32 = ffi.Library('user32', {
  'GetLastInputInfo': ['bool', ['pointer']]
});
```

**锁屏实现：**
- 创建全屏 BrowserWindow
- `alwaysOnTop: true`, `fullscreen: true`
- 使用 `globalShortcut` 拦截系统快捷键
- 使用 `setIgnoreMouseEvents(false)` 确保捕获所有输入

**系统托盘：**
```javascript
const { Tray, Menu } = require('electron');
const tray = new Tray('assets/icon.png');
```
