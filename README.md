# StandUpReminder - 久坐提醒工具

> 用 Trae + Electron，从零构建一款保护腰颈椎的健康提醒软件

## 📌 项目背景

当代年轻人，正在被「久坐」慢慢毁掉。

- 每天坐着超过 **8 小时**的办公族，腰椎长期处于压迫状态
- 久坐 1 小时，腰椎压力增加约 **40%**
- 长期前伸的坐姿，让颈椎每天额外承受 **10~15 斤** 的重量
- 中国超过 **2 亿人**存在颈椎亚健康问题，其中 30 岁以下占比超过 40%

传统的解决办法要么靠意志力，要么靠人工养生，但都难以坚持。**我们需要的，是一个真正能强制你站起来休息的工具。**

## 🎯 项目目标

最终交付物：**StandUpReminder** —— 一款会自动锁屏强制你休息的健康守护工具。

## 📸 功能介绍

### 1. 强制锁屏休息

到达设定时长后，应用将锁定你的屏幕，显示倒计时和健康小贴士。**无法跳过，无法忽视**——只有真正休息够了，才能继续工作。

### 2. 自定义健康提醒

不只是锁屏，应用还支持自定义提醒：喝水提醒（内置）、护眼提醒、运动提醒等，每个提醒独立倒计时，到点弹出通知。

### 3. 系统托盘后台运行

最小化到系统托盘，在后台默默守护。支持右键快捷操作：显示主窗口、开始/暂停监控、退出程序。

### 4. 推迟机制 + 警告提示

连续推迟 2 次后，窗口边缘会发出红色闪烁警告；第 3 次，系统将直接强制锁屏。

## 🚀 项目运行

### 环境要求

- Windows 10 及以上
- Node.js 18+
- Git

### 克隆项目

```bash
git clone https://github.com/fsan10/StandUpReminder.git
cd StandUpReminder
```

### 安装依赖

```bash
npm install
```

### 启动开发

```bash
npm start
```

### 打包发布（Windows）

```bash
npm run dist
```

打包完成后，安装包在 `dist/` 目录下，双击即可安装使用。

## 🔧 核心功能实现

### 1. 自定义提醒功能

支持多种健康提醒类型，每种提醒独立计时、独立配置：
- **喝水提醒**：内置默认，每 30 分钟提醒喝水
- **护眼提醒**：每 45 分钟提醒远眺放松
- **运动提醒**：每 60 分钟提醒起身活动
- **自定义提醒**：用户可自由添加新的提醒类型

每个提醒卡片显示名称、间隔时间、实时倒计时，支持开关、刷新、删除操作。

### 2. 锁屏强制机制

当工作时间到达，显示全屏遮罩窗口，`alwaysOnTop: true` + `skipTaskbar: true` 确保窗口置顶且无法通过任务栏切换。

### 3. 设置持久化

所有用户设置保存到本地 JSON 文件，下次打开应用自动恢复。

## 🔮 后续规划

### 1. 提醒语音包

支持多种语音提醒，不再只是单调的弹窗：
- 温柔女声、活力男声、可爱童声等多种语音风格
- 可自定义提醒语内容
- 支持调节音量、语速

### 2. 数据统计

记录每日休息次数、总休息时长、推迟次数等数据，生成健康报告。

### 3. 云同步

支持账号登录，跨设备同步设置和提醒数据。

## 💡 升华主题

**「久坐」是当代年轻人最大的健康隐患之一。**

很多人知道久坐不好，但就是改不掉。意志力靠不住，所以需要工具来帮助——这不是"矫情"，而是一种**对身体的主动管理**。

一款好的健康工具，不应该只是"提醒你"，而应该在你做不到的时候**替你做到**。StandUpReminder 的强制锁屏机制，就是这个理念的具体实现：**当你自己无法坚持，系统就介入帮你强制执行。**

> 技术的价值，不在于让你更高效地工作，而在于帮你更好地**生活**。
